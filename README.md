# DevHub

DevHub is a developer-focused social platform that combines the best parts of LinkedIn, Reddit, Instagram, and GitHub.

Core Mission:
Help developers learn, build, network, collaborate, and grow their careers.

Platforms:

* Web Application
* Android Application
* iOS Application

Main Features:

* Profiles
* Posts
* Communities
* Messaging
* Project Showcase
* GitHub Integration
* Learning Platform
* Job Board
* AI Features

Tech Stack:

* Next.js
* React Native (Expo)
* TypeScript
* Tailwind
* Shadcn UI
* Supabase
* PostgreSQL
* Vercel

Monetization:

* Premium Accounts
* Sponsored Jobs
* Advertisements
* Organization Pages
